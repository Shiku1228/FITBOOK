# FitBook 🏀

A Sports Facility & Coach Booking Platform built with Laravel 11.

FitBook connects athletes with sports facilities and coaches across the 
Philippines — search nearby courts, book time slots, hire coaches, and 
pay via GCash or card. All in one platform.

## Features
- 🔐 Role-based auth (Athlete, Coach/Owner, Admin) via Laravel Sanctum
- 🏟️ Facility & slot management with real-time availability via Firebase RTDB
- 📅 Booking system with automatic fee calculation
- 💳 Payments via PayMongo (GCash, Maya, Card) + Stripe
- 🤖 AI coach matching via OpenAI embeddings
- 🗺️ Nearby facility search via Google Maps API
- 📸 Media uploads via Cloudinary
- 🔔 Push notifications via Firebase FCM
- 🚀 CI/CD via GitHub Actions + Docker

## Tech Stack
Laravel 11 · MySQL · Firebase RTDB · Cloudinary · 
PayMongo · Stripe · OpenAI · Google Maps API · Docker
